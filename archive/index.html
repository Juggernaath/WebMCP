MCP Server for Browser Automation

AI Browser Control with Human Oversight

WebMCP lets Claude, GPT, and AI agents control your browser. When AI gets stuck on CAPTCHAs or needs help, it asks you — then continues automatically.

Add to Chrome How It Works
Claude: "Apply for the Python developer job on LinkedIn"
WebMCP: ✓ Navigated to linkedin.com/jobs
WebMCP: ✓ Found job listing, clicking Apply
WebMCP: ⚠ CAPTCHA detected — please solve it in your browser
You: *solves CAPTCHA*
WebMCP: ✓ Resuming... filling application form
WebMCP: ✓ Application submitted successfully!

Get Started in 3 Steps

Up and running in under 5 minutes.

1

Install the Chrome Extension

Add WebMCP to Chrome. The extension connects your browser to AI agents.

2

Install the MCP Server

Add the MCP server to your AI client (Claude Desktop, Cursor, etc.)

npx @webmcp/server
3

Start Automating

Ask your AI to browse the web. It handles automation, you handle the tricky parts.

"Go to amazon.com and find deals on headphones"
